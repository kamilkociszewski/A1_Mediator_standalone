# A1 Mediator for FlexRIC

## Run A1 Mediator
```
cd A1_Mediator
pip3 install -r requirements.txt
cd app
python3 main.py
(host_ip):9000/docs #to see available APIs
```

## Run C test app
```
flexric/build/examples/xApp/c/kpm_rc_A1
./xapp_kpm_rc_a1
```
